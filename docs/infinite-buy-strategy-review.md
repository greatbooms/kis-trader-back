# 무한매수법 전략 리뷰 — 장단점 및 개선 방향

## 무한매수법의 장점

- **감정 배제** — 규칙 기반이라 패닉셀/FOMO 방지
- **평단 하락 효과** — 떨어질수록 싸게 사니까 평단이 계속 내려감
- **반등 시 확실한 수익** — 평단 +10%만 되면 청산, 현실적 목표
- **단순함** — 복잡한 지표 분석 없이 가격+보유량만으로 동작

---

## 약점 (심각한 것들)

### 1. "결국 반등한다"는 전제가 위험

```
SOXL (3x 반도체 레버리지 ETF)
2021.12 고점: $74
2022.10 저점: $8  (-89%)

평단 +10% 회복하려면? → 원래 가격의 거의 복귀 필요
그 사이 40회차 넘겨서 수동 전환 → 사실상 물린 상태
```

레버리지 ETF는 **변동성 감쇠(volatility decay)** 때문에 횡보만 해도 가치가 녹는다. "장기 우상향"이 보장되지 않음.

### 2. 자본 효율이 나쁨

```
하락장: 매수만 체결, 매도는 안 걸림
→ $200 × 40회 = $8,000이 묶임
→ 그 동안 다른 기회에 투자 불가
→ 기회비용 큼
```

### 3. 손절이 없음

T ≥ 40에서 "수동 전환"이 유일한 안전장치인데, 이미 $8,000이 물린 상태. 진짜 안전장치라면 **손실 한도에서 잘라야** 하는데 그 로직이 없다.

### 4. 매도 구조의 문제

```
매일 보유량의 1/4 + 3/4를 매도 주문 → 체결 안 되면 다음날 또 걸음
→ 실제로는 소량 반등에 전량 매도될 수 있음
→ 이후 더 오르면 놓침 (수익 제한)
```

+10% 고정 목표는 **상승장에서 수익을 심하게 제한**한다.

### 5. 주문 방식 한계

LOC(장마감 지정가)만 사용 → 장중 급등/급락 대응 불가. 장중에 +15% 갔다가 종가에 -2%로 마감하면 매도 체결 안 됨.

## 개선 방향

### A. 손절 라인 추가
```
if (현재가 < 평단 × 0.7)  → 손절 (30% 손실 한도)
if (T ≥ 30 && 손익률 < -20%) → 포지션 축소
```

### B. 매도 목표를 동적으로
```
고정 +10% 대신:
- 단기(T<10): +5%에 1/3 매도
- 중기(T 10~20): +10%에 1/2 매도
- 장기(T>20): +15%에 전량 매도

또는 트레일링 스탑: 고점 대비 -5% 하락시 매도
```

### C. 종목 선별 기준 추가
```
현재: 아무 종목이나 넣으면 돌아감
개선:
- 200일 이동평균선 위에 있는 종목만 매수 허용
- RSI 30 이하 (과매도)일 때만 매수 강화
- 거래량 급증 시 매매 일시 중단
```

### D. 자본 관리
```
현재: 종목당 quota 고정
개선:
- 전체 포트폴리오의 N% 이상 한 종목에 투입 금지
- 총 투자금 대비 현금 비율 최소 30% 유지
- 동시 진행 종목 수 제한 (예: 최대 5개)
```

### E. 시장 상황 반영
```
현재: 시장 상황 무관하게 매일 매수
개선:
- VIX > 30: quota 절반으로 줄임 (공포 구간 방어)
- 주요 지수 200일선 아래: 신규 매수 중단
- 금리 급등기: 레버리지 ETF 매수 중단
```

## 결론

| 항목 | 평가 |
|------|------|
| 초보자 입문용으로 | **괜찮음** — 규칙이 단순하고 이해하기 쉬움 |
| 그대로 실전 투입 | **위험** — 손절 없음, 자본 효율 나쁨, 하락장 방어 부족 |
| 개선해서 쓴다면 | **쓸만함** — 손절 + 동적 매도 + 시장필터 추가하면 괜찮은 기반 |

무한매수법 자체를 버릴 필요는 없고, **기본 뼈대(분할매수 + 목표가 매도)는 유지하되 위의 안전장치를 추가**하는 게 현실적이다. 우리 프로젝트는 전략을 인터페이스로 분리해뒀으니, 기본 무한매수법을 먼저 구현하고 점진적으로 개선하는 방식이 좋다.
