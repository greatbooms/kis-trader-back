# 무한매수법 전략 설명

## 한 줄 요약
> **매일 정해진 금액만큼 분할 매수하되, 많이 빠지면 더 싸게 사고, 평단 +10%에 도달하면 판다.**

---

## 기본 규칙

### 1. 매일 같은 금액(quota)을 투자한다

예를 들어 quota = $200이면, 매일 $200어치씩 산다.

### 2. 진행률(T)로 현재 단계를 파악한다

```
T = 지금까지 투자한 총액 / quota

예: $200씩 넣는데 총 $2,000 투자했으면 → T = 10 (10회차)
```

### 3. T에 따라 매수 가격이 달라진다

| T (회차) | 기준 가격 (평단 대비) | 의미 |
|---------|---------------------|------|
| 0 | 평단 +10% | 처음엔 좀 비싸게라도 산다 |
| 10 | 평단 +5% | 점점 낮춰간다 |
| 20 | 평단 ±0% | 평단가에 산다 |
| 30 | 평단 -5% | 떨어져야 산다 |
| 40 | 평단 -10% | 많이 떨어져야만 산다 → 이후 수동 |

→ **초반에는 적극 매수, 후반에는 싸게만 매수** (물타기 방어)

---

## 매일 하는 일 (4단계)

$200 quota, 평단 $10, 100주 보유 상황이라고 가정:

```
① 매수1: 현재가 근처에서 N주 매수     ← "오늘치 물량 확보"
② 매수2: 평단 아래에서 M주 매수        ← "싸게 살 수 있으면 더 사자"
   (①+② 합쳐서 $200 이내)

③ 매도1: 보유량의 1/4 (25주)을 기준가에 매도  ← "일부 수익 실현"
④ 매도2: 나머지 3/4 (75주)를 평단+10%에 매도  ← "목표가 도달시 대량 청산"
```

**포인트**: 매수는 매일 체결되지만, 매도는 **가격이 올라야 체결**된다.

- 주가가 안 오르면 → 매수만 계속 (물량 축적)
- 주가가 오르면 → 매도 체결되면서 수익 실현

---

## 왜 "무한"매수법인가?

```
주가 하락 시:
  매수만 체결, 매도는 안 걸림
  → 물량이 계속 쌓임
  → 평단이 점점 내려감
  → 나중에 조금만 반등해도 +10%에 도달

주가 상승 시:
  매도 체결 → 수익 실현
  → 보유량 줄고 → T(진행률) 낮아짐
  → 다시 처음부터 매수 시작 (사이클 반복)
```

**떨어지면 사고, 오르면 파는 걸 끝없이 반복**한다고 해서 무한매수법.

---

## 주문 방식: LOC

원본 앱(InfiniteBuyStock)은 **LOC(Limit On Close)** 주문을 쓴다:
- 장 마감 시점에 "이 가격 이하면 사줘 / 이 가격 이상이면 팔아줘"
- 장중 변동에 흔들리지 않고, **종가 기준으로 깔끔하게 체결**
- 매일 장 마감 직전에 한 번만 실행하면 됨

---

## 안전장치

- **T ≥ 20**: 현재가 매수(Buy1) 중단 → 싼 가격 매수(Buy2)만
- **T ≥ 40**: 자동매매 완전 중단 → 수동 판단
- 40회차 = quota × 40 투자한 상태 → $200 기준 $8,000 투입, 이 이상은 리스크가 커서 사람이 판단

---

## 핵심 아이디어

```
"주가는 결국 어느 정도 반등한다"는 전제 하에,
떨어질 때 꾸준히 사서 평단을 낮추고,
평단 대비 +10%만 되면 파는 전략.

개별 종목이 아니라 SOXL, TQQQ 같은 레버리지 ETF에
주로 사용 (장기적으로 우상향하는 지수 기반).
```

## 수식 정리

```
입력:
  quota      = 일일 투자금액 (예: $200)
  avgPrice   = 매입평균가
  curPrice   = 현재가
  totalPrice = 매수총액 (= avgPrice × 보유수량)
  holdQty    = 보유수량

계산:
  T          = totalPrice / quota                    (진행 회차)
  baseRate   = (10 - T/2 + 100) / 100               (동적 비율, T=0→1.10, T=20→1.00)
  pivotPrice = baseRate × avgPrice                   (기준 가격선)

매수:
  buy1Price  = min(curPrice × 1.2, avgPrice)         (현재가 근처, T<20만)
  buy2Price  = pivotPrice - 0.01                     (기준선 바로 아래)
  buy1Qty, buy2Qty = 교대로 채워서 quota 이내

매도:
  sell1Price = pivotPrice                            (기준 가격선)
  sell1Qty   = round(holdQty / 4)                    (1/4 물량)
  sell2Price = avgPrice × 1.1                        (평단 +10%)
  sell2Qty   = holdQty - sell1Qty                    (3/4 물량)

특이사항:
  T ≥ 20 → buy1 생략, buy2만 실행
  T ≥ 40 → 자동매매 중단 (수동 전환)
```

### baseRate(동적 비율) 변화표

| T (진행률) | baseRate | pivotPrice (평단 $10 기준) |
|-----------|----------|--------------------------|
| 0 | 1.100 | $11.00 |
| 4 | 1.080 | $10.80 |
| 10 | 1.050 | $10.50 |
| 20 | 1.000 | $10.00 (= 평단) |
| 30 | 0.950 | $9.50 |
| 40 | 0.900 | $9.00 |

→ 초기에는 평단보다 높은 가격에 매도 (수익 실현), 후반에는 평단 아래에서 매수 (물타기)
